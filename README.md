# Deep Learning & PyTorch Learning Project

A collection of Jupyter notebooks documenting my learning journey through Deep Learning and PyTorch basics.

## Overview

This project contains hands-on learning materials covering fundamental concepts in Deep Learning and PyTorch, built with the help of Claude. Each notebook explores different aspects of neural networks and practical implementations.

## Project Structure

- **Notebooks** (01.ipynb - 017.ipynb): 17 sequential Jupyter notebooks covering various DL/PyTorch topics
- **data/**: Dataset storage directory (MNIST)
- **my_model.pth**: Pre-trained model weights
- **requirements.in**: Project dependencies

## Topics Covered

- Neural network fundamentals
- PyTorch basics and tensor operations
- Building and training models
- Image classification (MNIST)
- CNN architectures
- RNN/LSTM basics
- Model training and evaluation

## Getting Started

Install required dependencies:
```bash
pip install -r requirements.in
```

Then explore the notebooks in sequential order to follow the learning progression.

This is a learning project - the code is organized to demonstrate concepts as I learn them. Feel free to use this as a reference for your own DL/PyTorch journey!
